一些有用的数组方法

字符串和数组之间的转换

  • split() 将字符串转换为数组
    • '1,2,3,4,5'.split(',') ==> [1,2,3,4,5]
  • join() 将数组转换成字符串
    • [1,2,3,4,5].join(',') ==> '1,2,3,4,5'
  • toString() 将数组转换成字符串,但不局限于数组
    • ["Rocket","Flash","Bella","Slugger"].toString() ==> 'Rocket,Flash,Bella,Slugger'

添加和删除数组项

  • push() : 添加一个或多个要添加到数组的末尾元素
    • MyArray.push('你好','大家好')
  • pop() : 删除数组的最后一个元素
    • MyArray.pop()
  • unshift() : 添加一个元素到数组开头
  • shift() : 删除数组的第一个元素
