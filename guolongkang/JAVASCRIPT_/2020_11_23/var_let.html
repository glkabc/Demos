var 与 let 的区别

  • 变量提升不适用于let : 如果将下面例子中的 var 替换成 let 将不起作用并引起一个错误。
    • myName = 'Chris';

function logName() {
  console.log(myName);
}

logName();

var myName;
  • var 可以在使用时多次声明相同名称的变量,但是let不行 :下面将会出错,但是换成var不会
    • let myName = 'Chris';
let myName = 'Bob';

变量类型

  • Number

    • 你可以在变量中存储数字，不论这些数字是像30（也叫整数）这样，或者像2.456这样的小数（也叫做浮点数）。

    let myAge = 14;
  • String

    • 字符串是文本的一部分。当你给一个变量赋值为字符串时，你需要用单引号或者双引号把值给包起来，否则JavaScript将会把这个字符串值理解成别的变量名。

    let str = '你好';
  • Boolean

    • Boolean 的值有2种：true或false。它们通常被用于在适当的代码之后，测试条件是否成立。

    let runing = true;
  • Array

    • 数组是一个单个对象，其中包含很多值，方括号括起来，并用逗号分隔。

    let array = [1,2,3,4];
  • Object

    • 对象是现实生活中的模型的一种代码结构。

    let dog = { name : 'Spot', breed : 'Dalmatian' };
dog.name
