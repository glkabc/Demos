函数作用域和冲突

作用域（scope）在处理函数时是一个非常重要的概念。

  • 函数内定义的变量和其他东西都在它自己的单独范围内，意味着它们被锁在自己独立的隔间中, 不能被函数外的代码访问。
  • 所有函数的最外层被称为全局作用域。 在全局作用域内定义的值可以在任意地方访问。
相同的范围规则不适用于循环（for（）{...}）和条件块（if（）{...}） - 它们看起来非常相似，但它们不一样！小心不要让这些困惑。

函数内部的函数

我们可以在任意的地方去调用函数，甚至可以在另一个函数内部去调用另一个函数。

function one() {
    var value = one;
    two(value);
    three(value);
}
two(value) {
    console.log(value);
}
three(value) {
    console.log(value);
}
