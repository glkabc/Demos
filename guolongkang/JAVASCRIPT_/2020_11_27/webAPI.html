获取dom元素

推荐使用 Document.querySelector() 来获取页面上的节点

如果想对多个元素进行操作可以使用 Domcument.querySelectorAll()

对于获取元素引用，还有一些更旧的方法

  • Document.getElementById(): 选择一个id属性值已知的元素
    • var elementRef = document.getElementById('myId')
  • Document.getElementsByTagName(): 返回页面中包含的所有已知类型元素的数组。
    • var elementRefArray = document.getElementsByTagName('p')
  • ...

创建dom元素

  1. document.creatElement('p'): 创建一个P标签
    2. let p = document.createElement('p');
  2. p.textContent = 'hello': 为标签添加内容
    3. p.textContent = 'hello'
  3. parentNode.appendChild('p'): 将创建的标签添加到父节点里
    4. div.appendChild('p')

移动或删除元素

  • appendChild(): 添加元素
    • sect.appendChild(linkPara);
  • removeChild(): 删除元素
    • sect.removeChild(linkPara);

    要删除一个仅基于自身引用的节点可能稍微有点复杂，这也是很常见的。没有方法会告诉节点删除自己，所以你必须像下面这样操作。

    linkPara.parentNode.removeChild(linkPara);
